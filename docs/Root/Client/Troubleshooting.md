:::info
`WOOLY_DEBUG=1` enables verbose logging from the client to stderr. `WOOLYAI_DEBUG` is accepted as an alias (some automation used that name). Rebuild and use the same `libwooly.so` you get from `LD_PRELOAD` / `LIB_WOOLY_PATH`, or you will not see new log lines.
:::
